# SP500-Algorithmic-Scanner

## Overview

This project is a Python-based algorithmic scanner that downloads live financial data from Wall Street and evaluates trading signals based on the 20-period Exponential Moving Average (EMA 20). 

It is designed to automate market analysis, utilizing vectorization to generate instant Buy/Sell signals without relying on slow computational loops.

## Tech Stack

* **Language:** Python 3.x
* **Data Extraction:** `yfinance` (Live market API connection)
* **Data Processing:** `pandas` (Vectorized boolean indexing and EWMA calculations)
* **Data Visualization:** `matplotlib` (Automated technical charting)


## How it Works

1. Connects to the Yahoo Finance API to extract 6 months of daily S&P 500 data (`^GSPC`).
2. Calculates the EMA 20 dynamically using Pandas' exponential weighted moving average (`.ewm`) function, applying the traditional financial market adjustment formula.
3. Evaluates crossover conditions row-by-row to append actionable `Buy_Signal` and `Sell_Signal` columns.
4. Generates a professional technical chart highlighting price action vs. the moving average.
